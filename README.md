# 🧑‍🎓OTFK-ONTY-2KG-11-app
Цей репозиторій містить початкову фазу розробки проєкту NoteShare — платформи для швидкого обміну та структурування навчальних нотаток. Структура документа відображає логіку ЖЦ ПЗ (фаза ініціалізації та Discovery): для кого, навіщо, що є результатом та як він перевірятиметься.
## 🟩 Базовий рівень (Ініціалізація та Ядро цінності)
### 1. Лаконічний пітч (Elevator Pitch)
Для студентів, які не встигають конспектувати лекції або готуватися до іспитів самотужки, **NoteShare** — це вебплатформа, яка дозволяє миттєво ділитися структурованими конспектами та знаходити потрібні матеріали за тегами за 3 кліки. На відміну від хаотичних чатів у месенджерах, ми забезпечуємо єдину базу знань із рейтингом корисності.
### 2. Ядро цінності (Core Value Proposition)
* **Економія часу:** Пошук потрібної лекції займає менше хвилини завдяки системі тегів та фільтрів.
* **Якість контенту:** Краудсорсингова система оцінювання (лайки/рейтинги) виводить наверх лише якісні нотатки.
### 3. Стейкхолдери (Stakeholders)
* **Користувачі:** Студенти (активні автори контенту) та студенти (читачі/споживачі).
* **Адміністратори платформи:** Модератори контенту (для запобігання спаму та плагіату).
* **Власник продукту (Product Owner):** Команда розробки, що валідує ідею.
* ### 4. Вимірювана мета та ознаки успіху
* **Головна мета:** Залучити 100 активних користувачів з одного факультету протягом перших 2 тижнів після запуску MVP.
* **Перший набір ознак успіху:**
    * Успішно створено та налаштовано GitHub-репозиторій.
    * Опубліковано мінімум 20 перших навчальних матеріалів (база для старту).
    * Час завантаження однієї сторінки нотатки не перевищує 2 секунд.
